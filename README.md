# qlocky
Timer that could turn into a task manager in the future.
This application was created as a practice in the Qt Framework.

## How to build:
Run the following commands in terminal:
```bash
cmake -DCMAKE_BUILD_TYPE=Release . -B build/
cmake --build build/
```
